## Running a Game Session

### For the Host/Teacher:

#### Setup (on your computer):
1. Open **Admin Panel**
2. Click **"Create New Game"**
3. Share the game code with players

#### Display (on projector/TV):
1. Open **Main Display Screen** (can be same or different device)
2. Enter the game code
3. This will show the live leaderboard

#### Start Playing:
1. In Admin Panel, click **"Start Game"** when ready
2. Players can now guess letters!
3. Watch the leaderboard update in real-time

### For Players:

## Troubleshooting

**"Game not found"**
- Double-check the code
- Make sure admin created the game first

**"No words available"**
- Load initial words in Admin Panel
- Or add at least one custom word

**Players not showing up**
- Refresh Main Display Screen
- Make sure they entered correct code

**Game won't start**
- Need at least 1 word in database
- Check game status in Admin Panel

## Quick Reference

### Game Codes
- Always 6 characters
- Uppercase letters and numbers
- Unique for each game
- Share with players to join

### Scoring
- 100 points base
- -10 points per wrong guess
- Minimum 10 points per win

### Max Wrong Guesses
- 6 wrong guesses allowed
- Hangman has 6 body parts:
  1. Head
  2. Body
  3. Left arm
  4. Right arm
  5. Left leg
  6. Right leg
